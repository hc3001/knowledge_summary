#### 2、类型注解、类型推断
类型注解相当于手动注解类型，包括变量、函数传参/返回
类型推断是typescript根据现有的类型或赋值传参自动判断类型
```
var test1 = '22'
test1 = 11 //类型会报错，test1 已经根据类型推断出为string，再赋值为number不可行。

//可以这么解决
var test1: any = '22'
test1 = 11

var test2: (string | number) = '22'
test2 = 11
```
